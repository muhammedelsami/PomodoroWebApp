# Pomodoro Timer App

Modern ve kullanıcı dostu bir Pomodoro zamanlayıcı uygulaması. Çalışma sürenizi etkili bir şekilde yönetmenize ve görevlerinizi takip etmenize yardımcı olur.

## Özellikler

- 🕒 Özelleştirilebilir Pomodoro zamanlayıcı
- ✅ Görev yönetimi ve takibi
- 🌓 Karanlık/Aydınlık mod desteği
- 📊 Çalışma istatistikleri
- 🔔 Bildirim desteği
- 🎨 Farklı tema seçenekleri (Kırmızı, Mavi, Mor)

## Kurulum

1. Projeyi klonlayın:
```bash
git clone [repository-url]
cd pomodoro
```

2. Bağımlılıkları yükleyin:
```bash
npm install
```

3. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

## Kullanım

1. Çalışma ve mola sürelerini ayarlayın (varsayılan: 25 dakika çalışma, 5 dakika mola)
2. Başlat butonuna tıklayarak zamanlayıcıyı başlatın
3. Görev eklemek için "Add a new task" alanını kullanın
4. Tamamlanan görevleri işaretleyin
5. İstatistiklerinizi takip edin

## Teknolojiler

- Vite
- Tailwind CSS

- Hero Icons
